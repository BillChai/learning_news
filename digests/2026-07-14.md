2. **[AWS] CloudFormation 新增 Express mode,IaC 部署秒級回饋**
   為配合 AI agent 與開發者更快迭代,CloudFormation 推出 Express mode,可在數秒內取得部署確認並停用預設 rollback,支援立即修正重試。對 counter-cdk 這類 CDK/IaC 工作流是實用的加速選項。
   來源:[AWS — Top announcements of the AWS Summit New York 2026](https://aws.amazon.com/blogs/aws/top-announcements-of-the-aws-summit-in-new-york-2026/)|發布:2026-07

3. **[OpenAPI] Swagger 全面支援 OpenAPI 3.2.0,SmartBear 推出 API drift 偵測**
   Swagger 成為首個支援 OpenAPI 3.2.0 的開源工具鏈(UI/Client/Editor/ApiDOM),帶來 streaming API pattern、更豐富的 tag 階層;SmartBear 商用版也針對「AI 生成程式碼造成的 API drift」推出契約測試與漂移偵測。做 counter-swagger 規格維護值得留意。
   來源:[Swagger Blog — Support for OpenAPI 3.2.0](https://swagger.io/blog/swagger-launches-support-for-openapi-3-2-0/)|發布:2026(近期)

4. **[DB] PostgreSQL 19 Beta 1:原生 SQL/PGQ 圖查詢、線上 table repacking**
   6/4 釋出的 PG19 Beta 1 引入原生 SQL Property Graph Queries(SQL/PGQ),可在既有關聯式資料表上直接跑圖查詢,不用另建圖資料庫;另加入不停機回收儲存空間的 concurrent table repacking,正式版預計 9 月。
   來源:[PostgreSQL — PostgreSQL 19 Beta 1 Released!](https://www.postgresql.org/about/news/postgresql-19-beta-1-released-3313/)|發布:2026-06-04

5. **[DB] CloudNativePG 1.30.0:新增 DatabaseRole CRD,角色管理走向 GitOps**
