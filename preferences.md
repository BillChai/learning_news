# 偏好紀錄

這份檔案記錄使用者對每日新聞摘要的喜歡/不喜歡回饋，排程任務每次執行前會先讀取這裡，用來調整搜尋方向與比例。

目前涵蓋主題：

- 工程知識/教學型文章（約5則）：工具介紹、原理科普、「這個東西是怎麼運作的」、best practice，偏教育性內容
- 知名工程師個人部落格近期文章（1-2則）
- GitHub Trending 近期star數暴增的repo（1-2則）
- 工作技術棧：Kotlin/Java（後端）、AWS/AWS CDK（IaC）、OpenAPI/Swagger
- 更廣泛後端主題：AI/LLM 工程應用、資料庫（DB）、分散式系統/系統設計/架構
- 綜合 IT 新聞熱門話題

## 格式要求（每次執行都必須遵守）

- **連結一律指向特定文章 / 特定 repo，不可給列表頁或首頁。** 例如：GitHub Trending 項目要給該 repo 的網址（如 `github.com/owner/repo`），不是 `github.com/trending`；部落格要給該篇文章網址，不是部落格首頁；新聞要給該則報導/討論串，不是 news archive 或 HN 首頁。
- 使用者回饋一律追加進本檔（見下方回饋紀錄），之後每次執行都要遵守。
